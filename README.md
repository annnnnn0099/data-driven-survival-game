# data-driven-survival-game
Built a 2D survival game that tracks player behavior (movement, survival time, score) and uses data analytics to optimize gameplay.

# Data-Driven Survival Game

## Overview
A survival game built using Python and Pygame with integrated data analytics and machine learning.

## Features
- Player movement tracking
- Survival time analysis
- Dynamic difficulty using ML
- Data visualization

## Tech Stack
- Python
- Pygame
- Pandas
- Matplotlib
- Scikit-learn

## How to Run
pip install -r requirements.txt
python main.py
